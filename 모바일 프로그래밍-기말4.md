(**URI**를 통해 정보 공유)

**Intent putExtra() 메소드**

인텐트로 데이터를 주고 받을떄 사용하는 메소드

저장된 데이터는 key-value 형태로 관리



**양방향 액티비티**

putExtra() 로 출발 시키기 -> startActivityForResult() 로 전송 ->getExtra()로 받음

putExtra() 로 출발 시키기 -> setResult()로 전송 ->getExtra()로 받음



putExtra() - Intent 안에 데이터 넣기

startActivityForResult() - Intent를 가지고 Activity 실행하고 결과 기다리기

getIntExtra() - Intent 안의 데이터 꺼내기

**생명주기**



처음 실행:

onCreate() → onStart() → onResume()



다른 화면으로 이동:

onPause() → onStop()



다른 화면 갔다가 다시 돌아옴:

onRestart() → onStart() → onResume()



완전히 종료:

onPause() → onStop() → onDestroy()





메소드 뜻:

onCreate()  : 화면 처음 생성, 초기 설정

onStart()   : 화면이 보이기 시작

onResume()  : 사용자가 조작 가능

onPause()   : 잠깐 멈춤

onStop()    : 화면이 안 보임

onRestart() : 다시 돌아와 재시작

onDestroy() : 완전 종료
